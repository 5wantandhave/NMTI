## 一、产品定位

### 1.1 核心立意

- **自嘲是武器**：通过"牛马"标签（如"996死者""ATM牛马"），让用户哈哈大笑地认清自己"其实就是条牛马"，释放情绪
- **开心是目的**：结果页不只是吐槽，还嵌入正向彩蛋，帮助用户从自嘲中获得力量
- **情绪价值**：让打工人找到集体归属感——"原来大家都是牛马，我不是一个人在卷"
- **Slogan**："MBTI太体面，SBTI太抽象，NMTI直接给你牛马本马"

### 1.2 目标用户

- 18-35岁互联网打工人
- 熟悉996/内卷/摸鱼/PUA等网络梗
- 喜欢自嘲、需要情绪宣泄
- 活跃于微信/抖音/小红书

## 二、玩法机制

### 2.1 核心流程

```
首页引导 → 答题(40题) → 计算结果 → 展示结果页 → 逆天改命（彩蛋）
```

### 2.2 答题规则

- **题量**：40道三选一题目
- **时间**：约3-6分钟
- **形式**：单选卡片，点击即选中
- **进度**：顶部进度条显示
- **随机**：题目顺序每次随机打乱

### 2.3 评分系统

#### 4维度二分法

| 维度代码 | 维度名称 | 高分一侧 | 低分一侧 |
|----------|----------|----------|----------|
| **J/P** | 卷摸维度 | J-卷王规划牛 | P-摸鱼灵活马 |
| **I/E** | 耗放维度 | I-内耗社恐牛 | E-外放社牛马 |
| **F/T** | 付理维度 | F-感性付出牛 | T-理性工具马 |
| **S/N** | 实幻维度 | S-实干996牛 | N-幻想躺平马 |

#### 得分规则

```
每个选项给维度 +2 / +1 / 0 / -1 / -2 分
每维度约10道题覆盖，总分范围 -20 ~ +20
分数 > 0 → 取高分字母（J/I/F/S）
分数 ≤ 0 → 取低分字母（P/E/T/N）
```

#### 结果生成

```typescript
// 4字母组合 = 16种牛马类型
type = (scores.J > 0 ? 'J' : 'P')
     + (scores.I > 0 ? 'I' : 'E')
     + (scores.F > 0 ? 'F' : 'T')
     + (scores.S > 0 ? 'S' : 'N');
// 例：IJFS = 996死者卷王
```

### 2.4 重玩与彩蛋机制

- 结果页提供"再测一次"按钮
- 轻微随机（±1分/维度）增加结果变化可能
- 题目顺序每次随机
- **逆天改命**：查看"如果当初选择不同"的平行世界结果

## 五、技术架构

### 5.1 技术栈

| 类型 | 技术 |
|------|------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite |
| 样式 | Tailwind CSS |
| 图表 | SVG 原生绘制 |
| 状态 | React useState/useReducer |

### 5.2 项目结构

```
nmtbi/
├── src/
│   ├── components/
│   │   ├── Home.tsx         # 首页
│   │   ├── Quiz.tsx         # 答题页
│   │   ├── RadarChart.tsx   # 雷达图
│   │   └── Result.tsx       # 结果页（含逆天改命）
│   ├── data/
│   │   ├── questions.ts     # 40道题目
│   │   └── types.ts         # 16种类型（详细描述）
│   ├── hooks/
│   │   └── useQuiz.ts       # 答题逻辑
│   ├── types/
│   │   └── index.ts         # TS类型
│   ├── utils/
│   │   └── calculate.ts     # 评分算法 + 逆天改命计算
│   ├── App.tsx
│   ├── main.tsx
│   └── index.css
├── index.html
├── package.json
├── vite.config.ts
├── tailwind.config.js
└── tsconfig.json
```

### 5.3 数据流

```
questions.ts → useQuiz → Quiz.tsx → calculate.ts → Result.tsx
     ↓                                          ↑
  types.ts ─────────────────────────────────────┘
       ↓
  逆天改命计算 ───────────────────────────────→ AlternativeResult
```
