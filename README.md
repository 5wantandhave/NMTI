## 二、玩法机制

### 2.1 核心流程

```
首页引导 → 答题(16题) → 计算结果 → 雷达图展示 → 逆天改命
```

### 2.2 答题规则

- **题量**：16道四选一题目
- **时间**：约2分钟
- **形式**：单选卡片，点击即选中
- **进度**：顶部进度条显示
- **场景分组**：
  - 任务执行（1-4题）
  - 社交博弈（5-8题）
  - 突发压力（9-12题）
  - 利益分配（13-16题）

### 2.3 评分系统 - 常模均值切割法

#### 4维度连续体

| 维度代码 | 维度名称 | 高分 = 左侧字母 | 低分 = 右侧字母 |
|----------|----------|-----------------|-----------------|
| **J/P** | 卷摸维度 | J = 终极卷王 | P = 终极摸鱼 |
| **I/E** | 耗放维度 | I = 重度内耗 | E = 疯狂外放 |
| **F/T** | 付理维度 | F = 纯纯大冤种 | T = 绝对冷酷理智 |
| **S/N** | 实幻维度 | S = 脚踏实地 | N = 天天幻想躺平 |

#### 常模均值（社会基准）

| 维度 | 常模均值 μ | 说明 |
|------|-----------|------|
| JP | +2 | 社会普遍偏卷 |
| IE | +1 | 职场人普遍内耗 |
| FT | 0 | 感性理智各半 |
| SN | +2 | 打工被迫务实 |

#### 计分公式

```
用户原始分 X → Z-Score转化 → 与常模μ比较
X >= μ → 取左侧字母（J/I/F/S）
X < μ → 取右侧字母（P/E/T/N）
```

#### 维度对冲计算

```javascript
D_JP = ΣScore_J - ΣScore_P
D_IE = ΣScore_I - ΣScore_E
D_FT = ΣScore_F - ΣScore_T
D_SN = ΣScore_S - ΣScore_N

代码 = (D_JP >= μ_JP ? 'J' : 'P')
     + (D_IE >= μ_IE ? 'I' : 'E')
     + (D_FT >= μ_FT ? 'F' : 'T')
     + (D_SN >= μ_SN ? 'S' : 'N')
```

## 四、功能亮点

### 4.1 雷达图展示
- SVG 绘制四维雷达图
- 直观展示用户在卷/内耗/付出/务实四个维度的得分
- 显示原始分数（+分/-分）

### 4.2 逆天改命
- 选择任意一道题
- 切换到不同选项
- 实时看到牛马属性的"平行世界进化"
- 例如："如果你当初选了B，你将从 996死者卷王 进化为 精准摸鱼学家"

### 4.3 详细类型解读
- 核心特征标签
- 性格画像描述
